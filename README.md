# Практическая работа 2 Шаблоны проектов

Внутри архива два готовых шаблона.

Первый шаблон находится в каталоге csharp и содержит проект на C# с контейнером внедрения зависимостей и загрузкой модулей внутри сборки.

Второй шаблон находится в каталоге node и содержит проект на Node.js с простым контейнером, загрузчиком модулей из каталога и набором испытаний.

Оба шаблона включают диаграмму Mermaid и текст технического задания в каталоге docs, а также примеры модулей, которые можно расширять и заменять под свою тему.
